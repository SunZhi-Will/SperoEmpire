■ このフォントについて

JFドットナガ10シリーズは、「ナガ10」をアウトライン化された TrueType 形式のフォントに変換し、
Windows や Mac、Linux など幅広い環境で使用できるようにしたフォントのセットです。
フォントにはビットマップも含んでいます。
